### INT-01: HELIUS_RPC_URL Env Guard Gap

**Severity:** Medium | **Affected:** BADGE-03, BADGE-04, BADGE-05, BADGE-06

`HELIUS_RPC_URL` is used with non-null assertion `!` in two badge routes without Phase 14's guard pattern:
- `app/web/app/api/badges/claimed/route.ts:21` — caught by try/catch, returns empty array (UI safe)
- `app/web/app/api/badges/mint/route.ts:69,96` — after collection-address guard passes, unstructured 500

**Recommendation:** Add `if (!process.env.HELIUS_RPC_URL) return 503` to both routes; add `HELIUS_RPC_URL=` to `.env.local.example` badge section.
