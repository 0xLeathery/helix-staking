## Cross-Phase Integration Report

Integration checker identified **2 gaps** out of ~25 integration points (92% complete):

### Integration Gap 1: BpdAborted Event Not Indexed

**Severity:** MEDIUM
**Files affected:**
- `services/indexer/src/types/events.ts` -- missing BpdAborted interface
- `services/indexer/src/worker/processor.ts` -- missing case handler
- `services/indexer/src/db/schema.ts` -- missing bpd_aborted_events table

**Impact:** When authority calls `abort_bpd`, the event is emitted on-chain but the indexer skips it (unknown event type). No audit trail for emergency abort actions.

### Integration Gap 2: Frontend IDL Out of Sync

**Severity:** MEDIUM
**Files affected:**
- `app/web/public/idl/helix_staking.json` -- has 13/14 instructions (missing abort_bpd)
- `app/web/types/program.ts` -- HelixStaking type missing abortBpd method

**Impact:** Authority cannot call `abort_bpd` from the web interface. Must use CLI for emergency operations.

**Root cause:** IDL was copied to frontend before `abort_bpd` was added in Phase 8. No automated sync step.
