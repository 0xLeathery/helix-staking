<objective>
Add anti-whale mechanics to prevent large stakers from dominating BPD distribution and accruing disproportionate T-share bonuses.

Purpose: Without whale limits, a single large actor can capture the majority of the BPD pool and receive outsized T-share bonuses. This undermines fairness for retail participants and creates a game-theoretic incentive for whale concentration.
Output: (1) BPD per-stake cap at 5% of pool, (2) BPB diminishing returns above 150M HELIX.
</objective>

  <action>
    Modify `calculate_bpb_bonus` to use tiered calculation:

    ```rust
    /// Calculate Bigger Pays Better (BPB) bonus with diminishing returns.
    ///
    /// Tier 1: 0 → BPB_THRESHOLD (150M):      Linear 0 → 1.0x (unchanged)
    /// Tier 2: BPB_THRESHOLD → BPB_TIER_2 (500M): Linear 1.0x → 1.25x (25% slope)
    /// Tier 3: BPB_TIER_2 → BPB_TIER_3 (1B):      Linear 1.25x → 1.4x (15% slope)
    /// Tier 4: Above BPB_TIER_3:                    Hard cap at BPB_MAX_BONUS (1.5x)
    pub fn calculate_bpb_bonus(amount: u64) -> Result<u64> {
        if amount == 0 {
            return Ok(0);
        }

        // Tier 1: Linear 0 → PRECISION (unchanged from original)
        if amount <= BPB_THRESHOLD {
            // Original formula: (amount / 10) * PRECISION / (threshold / 10)
            return Ok(
                (amount as u128)
                    .checked_mul(PRECISION as u128)
                    .ok_or(error!(HelixError::Overflow))?
                    .checked_div(BPB_THRESHOLD as u128)
                    .ok_or(error!(HelixError::Overflow))?
                    as u64
            );
        }

        // Base bonus at threshold = 1.0x = PRECISION
        let mut bonus: u128 = PRECISION as u128;

        // Tier 2: 25% additional from threshold to tier 2
        if amount <= BPB_TIER_2 {
            let excess = (amount - BPB_THRESHOLD) as u128;
            let tier_range = (BPB_TIER_2 - BPB_THRESHOLD) as u128;
            let tier_bonus = 250_000_000u128; // 0.25x in PRECISION
            bonus = bonus.checked_add(
                excess.checked_mul(tier_bonus).ok_or(error!(HelixError::Overflow))?
                    .checked_div(tier_range).ok_or(error!(HelixError::Overflow))?
            ).ok_or(error!(HelixError::Overflow))?;
        } else {
            // Full tier 2 bonus
            bonus = bonus.checked_add(250_000_000u128).ok_or(error!(HelixError::Overflow))?;

            // Tier 3: 15% additional from tier 2 to tier 3
            if amount <= BPB_TIER_3 {
                let excess = (amount - BPB_TIER_2) as u128;
                let tier_range = (BPB_TIER_3 - BPB_TIER_2) as u128;
                let tier_bonus = 150_000_000u128; // 0.15x in PRECISION
                bonus = bonus.checked_add(
                    excess.checked_mul(tier_bonus).ok_or(error!(HelixError::Overflow))?
                        .checked_div(tier_range).ok_or(error!(HelixError::Overflow))?
                ).ok_or(error!(HelixError::Overflow))?;
            } else {
                // Above tier 3: hard cap
                bonus = BPB_MAX_BONUS as u128;
            }
        }

        // Safety cap
        let final_bonus = bonus.min(BPB_MAX_BONUS as u128);
        Ok(u64::try_from(final_bonus).map_err(|_| error!(HelixError::Overflow))?)
    }
    ```
