**instructions/unstake.rs** -- Create the complete unstake instruction:

**Account struct `Unstake`:**
- `user`: Signer, mut
- `global_state`: Account<GlobalState>, mut, seeds=[GLOBAL_STATE_SEED], bump=global_state.bump
- `stake_account`: Account<StakeAccount>, mut, seeds=[STAKE_SEED, user.key().as_ref(), &stake_account.stake_id.to_le_bytes()], bump=stake_account.bump, constraint: has_one=user @ HelixError::UnauthorizedStakeAccess, constraint: stake_account.is_active @ HelixError::StakeAlreadyClosed
- `user_token_account`: InterfaceAccount<TokenAccount>, mut, associated_token::mint=mint, associated_token::authority=user, associated_token::token_program=token_program
- `mint`: InterfaceAccount<Mint>, mut, seeds=[MINT_SEED], bump
- `mint_authority`: UncheckedAccount, CHECK: PDA mint authority, seeds=[MINT_AUTHORITY_SEED], bump=global_state.mint_authority_bump
- `token_program`: Program<Token2022>

**Handler `unstake(ctx)`:**
1. Get clock: `Clock::get()?`
2. Calculate pending rewards: `calculate_pending_rewards(stake.t_shares, global_state.share_rate, stake.reward_debt)?`
3. Determine penalty:
   - If `clock.slot < stake.end_slot`: EARLY unstake
     - `penalty = calculate_early_penalty(stake.staked_amount, stake.start_slot, clock.slot, stake.end_slot)?`
   - Else: ON-TIME or LATE unstake
     - `penalty = calculate_late_penalty(stake.staked_amount, stake.end_slot, clock.slot, global_state.slots_per_day)?`
4. Calculate return amount: `staked_amount - penalty` (checked_sub)
5. Total mint to user: `return_amount + pending_rewards` (checked_add). Penalties are burned (tokens never minted = burned). Rewards are new inflation tokens minted.

6. **CRITICAL: Mark inactive BEFORE CPI** -- `stake.is_active = false`
7. Update GlobalState:
   - `total_unstakes_created += 1`
   - `total_tokens_unstaked += stake.staked_amount` (track original amount)
   - `total_shares -= stake.t_shares` (checked_sub)
   - `total_tokens_staked -= stake.staked_amount` (checked_sub, reduces active staked amount)
8. Mint tokens to user via `anchor_spl::token_2022::mint_to()` CPI with PDA signer (mint_authority seeds + bump):
   ```rust
   let authority_seeds = &[MINT_AUTHORITY_SEED, &[global_state.mint_authority_bump]];
   let signer_seeds = &[&authority_seeds[..]];
   ```
   Mint `return_amount + pending_rewards` in a single mint_to call.
   If total mint amount is 0 (100% penalty, no rewards), skip the CPI.
9. Emit `StakeEnded` event with penalty_type: 0=None, 1=Early, 2=Late

**Update instructions/mod.rs** -- Add `pub mod unstake; pub use unstake::*;`

**Update lib.rs** -- Add instruction endpoint:

