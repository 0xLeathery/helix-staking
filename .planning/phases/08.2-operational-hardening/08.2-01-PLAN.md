## Tasks

### Task 1: Remove NEXT_PUBLIC_TEST_WALLET_SECRET from client bundle (F-01 — CRITICAL)

**File:** `app/web/app/providers.tsx`

1. Remove any reference to `process.env.NEXT_PUBLIC_TEST_WALLET_SECRET`
2. If test wallet functionality is needed, move it behind a server-side check:
   - Create `app/web/app/api/test-wallet/route.ts` that returns the test wallet only in development
   - Or use a Playwright-style cookie/header check on the server side
   - Or simply remove test wallet auto-connect and require manual wallet connection
3. Ensure no other `NEXT_PUBLIC_` env vars contain sensitive material — audit all `.env*` files

**File:** `app/web/.env.local` (and any `.env*` files)
1. Remove `NEXT_PUBLIC_TEST_WALLET_SECRET` entirely
2. If needed for dev, use a non-NEXT_PUBLIC_ prefixed name and access server-side only

**Post-fix:** Rotate the leaked keypair immediately (it should never be used for holding real funds)

### Task 2: Add HSTS header (F-07)

**File:** `app/web/middleware.ts`

Add to the response headers alongside existing CSP headers:
```typescript
response.headers.set(
  "Strict-Transport-Security",
  "max-age=63072000; includeSubDomains; preload"
);
```

### Task 3: Remove devnet RPC default in production (OPS-13)

**File:** `services/indexer/src/lib/env.ts`

Change the `RPC_URL` field to not have a `.default()` value:
```typescript
// Before:
RPC_URL: z.string().url().default("https://api.devnet.solana.com"),

// After:
RPC_URL: z.string().url().describe("Required: Solana RPC endpoint URL"),
```

Add a `CLUSTER` env var for validation:
```typescript
CLUSTER: z.enum(["devnet", "mainnet-beta", "localnet"]).default("devnet"),
```

### Task 4: Clean production scripts (OPS-02 partial)

**Files:** `scripts/devnet-validate.ts`, `scripts/devnet-validate-bpd.ts`, `scripts/devnet-validate-claims.ts`

1. Add comments marking raw keypair loading as DEVNET-ONLY
2. Add a cluster check at top of scripts — refuse to run against mainnet:
```typescript
const cluster = process.env.CLUSTER || "devnet";
if (cluster === "mainnet-beta") {
  throw new Error("SAFETY: Do not run validation scripts against mainnet with file-based keypairs. Use Squads multisig.");
}
```
