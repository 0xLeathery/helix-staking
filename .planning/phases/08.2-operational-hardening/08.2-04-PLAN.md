### Task 1: Add is_paused flag to GlobalState

**File:** `programs/helix-staking/src/state/global_state.rs`

Use reserved byte for the pause flag to avoid reallocation (same pattern as BPD window flag):

```rust
impl GlobalState {
    // reserved[0] = BPD window active (existing)
    // reserved[1] = is_paused flag (NEW)

    pub fn is_paused(&self) -> bool {
        self.reserved[1] == 1
    }

    pub fn set_paused(&mut self, paused: bool) {
        self.reserved[1] = if paused { 1 } else { 0 };
    }
}
```

### Task 2: Create pause/unpause instructions

**File:** `programs/helix-staking/src/instructions/pause.rs` (NEW)

```rust
use anchor_lang::prelude::*;
use crate::state::GlobalState;
use crate::errors::HelixError;

#[derive(Accounts)]
pub struct Pause<'info> {
    #[account(mut, seeds = [b"global_state"], bump)]
    pub global_state: Account<'info, GlobalState>,
    #[account(constraint = authority.key() == global_state.authority @ HelixError::Unauthorized)]
    pub authority: Signer<'info>,
}

pub fn handler(ctx: Context<Pause>) -> Result<()> {
    ctx.accounts.global_state.set_paused(true);
    msg!("Program paused by authority");
    Ok(())
}
```

### Task 5: Add BPD lifecycle health endpoint (OPS-08)

**File:** `services/indexer/src/api/routes/health.ts`

Add `/health/bpd` route:

```typescript
app.get("/health/bpd", async (req, reply) => {
  // Fetch ClaimConfig from on-chain to check BPD state
  // If bpd_window_active has been true for > 1 hour, report degraded
  // If bpd_stakes_distributed < bpd_stakes_finalized for > 30 minutes, report stalled

  const claimConfig = await fetchClaimConfig(connection);
  if (!claimConfig) {
    return reply.send({ status: "ok", bpdActive: false });
  }

  const windowActive = claimConfig.reserved[0] === 1; // BPD window flag
  const stakesFinalized = claimConfig.bpdStakesFinalized;
  const stakesDistributed = claimConfig.bpdStakesDistributed;

  const status = {
    bpdActive: windowActive,
    stakesFinalized,
    stakesDistributed,
    calculationComplete: claimConfig.bpdCalculationComplete,
    remaining: stakesFinalized - stakesDistributed,
    health: windowActive && stakesDistributed < stakesFinalized ? "in_progress" : "ok",
  };

  reply.send(status);
});
```
