       - Limit to last 365 days by default (optional query param for limit).
    
    2. `GET /api/stats/distribution/stakes`:
       - Query `stake_created_events`.
       - Return histogram data for stake durations.
       - Buckets: < 30 days, 30-90, 90-180, 180-365, 365-730, > 730.
       - Return count and totalAmount per bucket.
       - Note: Requires raw SQL or careful Drizzle aggregation since buckets aren't columns. If raw SQL is hard with Drizzle, fetch all active stakes (id, days, amount) and aggregate in memory (acceptable for < 100k stakes, but SQL `CASE WHEN` is preferred).
       - Use `sql` template tag from drizzle-orm for `CASE WHEN` aggregation.
    
    Update the `statsRoutes` plugin to register these.
  </action>
