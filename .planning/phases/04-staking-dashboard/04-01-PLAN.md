    - "use client" component wrapping: ConnectionProvider (endpoint="/api/rpc") -> WalletProvider (wallets=[Phantom, Solflare], autoConnect=false per security requirement) -> WalletModalProvider -> QueryClientProvider (staleTime: 15_000, gcTime: 30_000, refetchOnWindowFocus: true, retry: 3) -> ThemeProvider (attribute="class", defaultTheme="dark", forcedTheme="dark", enableSystem=false)

    - QueryClient instantiated OUTSIDE component (singleton)

    - Inter font from next/font/google
    - Metadata: title "HELIX Staking", description "Stake HELIX tokens to earn T-Shares and daily rewards"

    **5. CSP middleware (`middleware.ts`):**
    - Generate nonce per request via crypto.randomUUID()
    - CSP header: default-src 'self'; script-src 'self' 'nonce-{nonce}' 'strict-dynamic' (add 'unsafe-eval' in dev only); style-src 'self' 'unsafe-inline' (needed for wallet adapter styles); img-src 'self' blob: data:; font-src 'self'; object-src 'none'; base-uri 'self'; form-action 'self'; frame-ancestors 'none'; connect-src 'self' (wallets need this); upgrade-insecure-requests
    - Set X-Frame-Options: DENY, X-Content-Type-Options: nosniff
    - Matcher: exclude _next/static, _next/image, favicon.ico

    **6. RPC proxy (`app/api/rpc/route.ts`):**
    - POST handler that reads HELIUS_RPC_URL from process.env
    - Forwards JSON-RPC body to Helius, returns response
    - Error handling: 500 with generic "RPC request failed" message
    - Rate limiting: not needed for MVP (Helius handles it)

    **7. Home page (`app/page.tsx`):**
    - Server component with "Connect your wallet to start staking" hero
    - Import and render WalletButton component (created in Task 2)
    - Show protocol name "HELIX" and brief tagline

    **8. Environment:**
    - `.env.local.example` with `HELIUS_RPC_URL=https://mainnet.helius-rpc.com/?api-key=YOUR_KEY`
